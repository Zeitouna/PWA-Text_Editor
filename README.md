# PWA-Text_Editor

## Description

Build an off and online text editor.

## User Story

```md
AS A developer
I WANT to create notes or code snippets with or without an internet connection
SO THAT I can reliably retrieve them for later use
```

## Installation

Clone this repo and run `npm i`
